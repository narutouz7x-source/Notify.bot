# Instagram Notification Discord Bot (v14)

A professional Node.js Discord bot that automatically tracks a public Instagram profile using Apify's scraping network and sends real-time rich embeds to a designated channel. It also features modern Slash Commands (`/`) for interactivity.

---

## Features

* 🔄 **Automated Tracking:** Checks a public Instagram profile every 15 minutes.
* 🖼️ **Rich Media Embeds:** Delivers the newest post or reel, including captions and the full image cover layout.
* 📂 **Local Caching:** Uses a local storage system to prevent duplicate notifications.
* ⚡ **Popup Commands:** Utilizes Discord's global application Slash Commands (`/help`).
* 🏷️ **Mention Listener:** Automatically detects when the bot is tagged and redirects users to the slash menu.

Prerequisites
​Before running the bot, you will need:
​Node.js (v16.11.0 or higher recommended)
​A Discord Bot Token and Client ID from the Discord Developer Portal.
​An Apify API Token from your Apify Console settings page under the Integrations tab.
​Installation & Setup
​1. Install Dependencies
​Open your terminal in the project directory and run the following command to install the required modules:

<npm install discord.js axios dotenv>


2. Configure Environment Variables
​Create a file named .env in the root directory of your project and fill in your details
[DISCORD_TOKEN=your_discord_bot_token_here]
[CLIENT_ID=your_bot_application_id_here]
[TARGET_CHANNEL_ID=your_discord_channel_id_here]
[INSTAGRAM_USERNAME=instagram_account_to_track]
[APIFY_API_TOKEN=your_apify_personal_token_here]
